# linked-list Using C++
Simple c++ Program to show simple linked list operations
### Insert new node.
### Insert node in the midile.
### Delete node.
### print linked list.


## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
